# SPI Projekt Zaliczeniowy

## Opis projektu
Projekt dotyczy aplikacji pomagającej studentom planować naukę. Rozwiązuje problem braku struktury i przeładowania informacjami na studiach. Odbiorcą są osoby 18–25 lat, które chcą lepiej organizować zadania. Na pewno znajdą się: lista zadań, kalendarz tygodniowy, proste statystyki.

## Zawartość projektu
- `src/opis_projektu.txt`: Zawiera krótki opis projektu.
- `src/diagram_flowchart.txt`: Zawiera prosty diagram ASCII lub JPG przedstawiający strukturę projektu.
- `src/inspiracje.txt`: Zawiera inspiracje, listę elementów projektu, wstępne dane wejściowe oraz źródła pomysłu.

## Instrukcje
Aby uruchomić projekt, należy:
1. Skopiować pliki do lokalnego katalogu.
2. Otworzyć plik `src/opis_projektu.txt`, aby zapoznać się z celem projektu.
3. Przejrzeć `src/diagram_flowchart.txt`, aby zrozumieć strukturę aplikacji.
4. Sprawdzić `src/inspiracje.txt` w celu poznania inspiracji i pomysłów, które przyczyniły się do stworzenia projektu.
